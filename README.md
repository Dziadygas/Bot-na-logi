# 📝 Discord Logger Bot

## 🌟 Funkcje
- Logowanie usuniętych wiadomości
- Logowanie edytowanych wiadomości
- Logowanie banów i odbanowań
- Wszystko w formie ładnych embedów

## 📋 Wymagania
- Node.js v16+
- Discord.js v14
- Uprawnienia bota:
  - `View Channel`
  - `Read Message History`
  - `Ban Members` (do logowania banów)
